# 🛒 ShopKart - E-Commerce Shopping Cart

A modern, feature-rich shopping cart application built with **vanilla JavaScript** (ES6+), HTML5, and CSS3.

![ShopKart Preview](preview.png)

## ✨ Features

### 🎯 Core Features
- ✅ Product catalog with 10+ items
- ✅ Real-time cart management
- ✅ Add/Remove/Update cart items
- ✅ Automatic price calculations
- ✅ Tax & shipping calculations
- ✅ Member discounts (Premium/Regular)
- ✅ Coupon code support
- ✅ Responsive design (Mobile/Tablet/Desktop)

### 🔍 Advanced Features
- ✅ Product search
- ✅ Category filtering
- ✅ Price range filtering
- ✅ Rating filtering
- ✅ Multiple sort options
- ✅ Local storage persistence
- ✅ Toast notifications
- ✅ Smooth animations
- ✅ Modal dialogs
- ✅ Order confirmation

### 💳 Checkout Features
- ✅ Order summary
- ✅ Customer information
- ✅ Payment method selection
- ✅ Order ID generation
- ✅ Delivery date estimation
